# ClassMap / Kindle Minds — Claude Code Entry Point

This is a multi-app portfolio repo (Next.js 16 + React 19 + Tailwind 4 + shadcn/ui) developed by a **3-agent coordination protocol** (A = PM, B = developer, C = QA). It deploys to GitHub Pages as a static prototype and runs locally as a full-stack app with live AI via the `claude` CLI.

When you start a session in this directory, read these in order:

1. **`docs/CONTINUATION.md`** — "I'm sitting down at this repo, what's the current state and what do I do next?"
2. **`docs/PROJECT_STATE.md`** — every task across all phases, what's done, what's left
3. **`docs/3_AGENT_PROTOCOL.md`** — how the A/B/C protocol works
4. **`docs/memory/`** — agent memory snapshots (user preferences, conventions, gotchas)
5. **`coordination/`** — live coordination state (`BACKLOG.md`, `HANDOFFS.md`, `ISSUES.md`)
6. **`coordination/AGENT_B_PROMPT.md`** / **`AGENT_C_PROMPT.md`** — what to paste into B's and C's sessions when bringing the team up

For Next.js 16 conventions and the Next-specific warning, see `AGENTS.md` (auto-imported below).
